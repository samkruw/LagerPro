🔧 LagerPro einrichten

Firebase Console → Projekteinstellungen → Deine Apps → Config
Werte einzeln aus dem Config-Block unten abtippen oder einfügen.

Firebase Config (Referenz):
apiKey: "AIza...",
authDomain: "projekt.firebaseapp.com",
projectId: "projekt-id",
storageBucket: "projekt.appspot.com",
messagingSenderId: "123456789",
appId: "1:123:web:abc"
Nur Kleinbuchstaben, Zahlen, Bindestriche. Alle Nutzer desselben Betriebs verwenden dieselbe ID.
💡 Firestore Security Rules — einmalig in Firebase Console → Firestore → Regeln einfügen:

rules_version = '2';
service cloud.firestore {
 match /databases/{db}/documents {
  match /kunden/{kid}/{col}/{doc} {
   allow read, write: if request.auth != null;
  }
 }
}
Werte werden nur lokal im Browser gespeichert · zurücksetzen
